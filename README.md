# workshop_risc_performance

## Requirements

> [!NOTE]  
> The following utilities and Python libraries must be installed:


- `free`
- `atop`
- Python libraries:
  - `numpy>=1.26.4`
  - `matplotlib>=3.9.2`

## Running the script

To run the script, use the following command:

```bash
./setup_and_run.sh
```
